# vCluster Scenarios

Interactive Killercoda scenarios for learning vCluster.

| Scenario | Title | Description |
|----------|-------|-------------|
| 101 | Basic vCluster Operations | Install CLI, create/inspect/deploy/cleanup vClusters |
| 102 | Multi-Tenancy Isolation | Multiple vClusters with same app names, port-forwarding |
| 103 | Advanced CRD Management | CRD isolation between host and vClusters |
| 104 | Advanced Configuration | vcluster.yaml config, resource limits, sync, monitoring |
| 105 | Backup and Disaster Recovery | YAML export/import, data store backup, restore |
| 106 | Tenancy Models | Shared, dedicated, and private node configurations |
